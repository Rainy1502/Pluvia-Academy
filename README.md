# Pluvia Academy — Scaffold

Ini adalah kerangka awal situs Pluvia Academy (kursus online programmer).

Apa yang ada di folder ini:

- `index.html` — Halaman utama (statis) dengan header, hero, daftar kursus, paket, dan footer.
- `css/style.css` — Styles dasar dan responsif.
- `js/app.js` — JavaScript minimal (nav toggle, update tahun).
- `assets/img/placeholder.svg` — Placeholder ilustrasi.

Cara menjalankan:

1. Buka `index.html` di browser (cukup double-click atau buka file melalui Live Server di VS Code).

Rekomendasi langkah selanjutnya:

- Ganti placeholder dengan gambar/ikon dari Figma.
- Tambah halaman detail kursus, login, dashboard siswa.
- Integrasikan templating (EJS/Handlebars) atau framework (React/Vue) jika ingin interaktivitas.
- Siapkan struktur API/backend (Node/Express) untuk pembayaran, otentikasi, dan manajemen kursus.

Jika mau, saya bisa lanjut membuat halaman login, routing dasar, atau starter Express/Next.js untuk proyek ini.
