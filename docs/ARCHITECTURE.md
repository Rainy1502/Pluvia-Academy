# Architecture

This scaffold follows a simple Express + Handlebars layout similar to the provided Weather Website:

- `public/` — static assets (css, js, img)
- `src/` — application code (server entry point)
- `templates/` — handlebars templates (partials + views)
- `tests/` — basic tests
- `scripts/` — helper scripts

Next steps: wire data sources, connect templates to `src/utils` data, and add CI/test scripts.
